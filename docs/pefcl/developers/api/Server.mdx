## Exports

PEFCL exposes a few useful exports that be used from other resources to integrate better with the financing system.
The exports support both callback & promises.

<ExportComponent
  name="addBankBalance"
  luaExample={'exports.pefcl:addBankBalance(source, amount, message)'}
  jsExample={'await exports.pefcl.addBankBalance(source, amount, message)'}
  tsExample={
    'await exports.pefcl.addBankBalance(source: number, amount: number, message?: string)'
  }
  desc="Adds money to the default account of the player"
/>

- Export arguments:
  - source: `number`
    - PlayerSrc
  - amount: `number`
    - Amount of money to be added to default the account
  - message: `string`
    - Transaction message

<ExportComponent
  name="createInvoice"
  luaExample={'exports.pefcl:createInvoice(source, from, amount, message)'}
  jsExample={'await exports.pefcl.createInvoice(source, from, amount, message)'}
  tsExample={
    'await exports.pefcl.createInvoice(source: number, from: string, amount: number, message: string)'
  }
  desc="Creates an invoice"
/>

- Export arguments:
  - source: `number`
    - PlayerSrc
  - from: `string`
    - This is a message that should explain who sent the invoice. Examples:
      - "Police Dpt."
      - "John Doe"
  - amount: `number`
    - Decides how much the invoice should be
  - message: `string`
    - Invoice description. Examples:
      - "For the repairs"
